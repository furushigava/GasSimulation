# GasSimulation
## Problem Set 

Современная равновесная термодинамика строится аксиоматически, то есть на основе конечного числа постулатов, имеющих эмпирическое подтверждение — начал термодинамики. Их количество и формулировки варьируются в зависимости от автора, однако в подавляющем большинстве случаев выделяют четыре начала термодинамики: положение о существовании транзитивного равновесного состояния, закон сохранения энергии, закон неубывания энтропии и теорема Нернста. Определённое обоснование этим постулатам и экспериментальным фактам даёт статистическая физика, предлагающая вероятностный подход к описанию систем из большого числа частиц. Её успехи связаны с тем, что несмотря на рост сложности и запутанности поведения системы при увеличении числа частиц, при очень большом числе частиц появляются новые статистические закономерности. Таким образом, например, обосновывается связь между микроскопическими параметрами (размеры, масса, геометрия молекул) и макроскопическими (теплоёмкость, энтропия, критические явления), которая при чисто аксиоматическом подходе неочевидна. Основой статистического подхода являются понятия фазового пространства и функций распределения. В качестве аксиомы обычно принимается эргодическая гипотеза, утверждающая равенство средних значений физических величин по времени и по ансамблю. Эргодичность также приводит к свойству систем «забывать» начальные условия в процессе прихода к равновесию. В рамках этого проекта предлагается методами молекулярной динамики смоделировать поведение системы из большого числа классических частиц, взаимодействующих между собой и с окружением, и проверить выполнение следующих положений: 
1. Распределение Максвелла. 
2. Распределение Больцмана при наложении внешнего потенциального поля. 
3. Эргодическая гипотеза — сходимость временных и ансамблевых средних и «забывание» начальных условий. 
4. Сохранение энергии в изолированной системе (первое начало термодинамики). 
5. Неубывание энтропии (второе начало термодинамики). 
6. Модель случайных блужданий броуновской частицы и соотношение Эйнштейна: $\langle r^2\rangle \propto n T t$, где $n$ — размерность пространства блуждания, а $T$ определяется из распределения Максвелла. 
7. Влияние внутренней структуры молекулы (многоатомность, линейность) на перечисленные выше пункты.
8. Учет потенциалов Леннард-Джонса, Морзе и Дерягина–Ландау–Фервея–Овербека.
