## Параметры симуляции

Основные параметры области симуляции и временны́х шагов.

| Параметр | По умолчанию | Диапазон | Единицы | Описание |
|----------|--------------|----------|---------|----------|
| `width` | 500 | 100–2000 | px | Ширина области симуляции |
| `height` | 500 | 100–2000 | px | Высота области симуляции |
| `dt` | 0.01 | 0.001–0.1 | у.е. | Шаг времени симуляции |
| `stats_interval` | 0.1 | 0.01–1.0 | с | Интервал обновления статистики |

**Примечания**:
- Меньший `dt` даёт более точное моделирование, но замедляет симуляцию
- `stats_interval` влияет на частоту обновления графиков

## Параметры изменения состояния

Скорости процессов нагрева, охлаждения, расширения и сжатия.

| Параметр | По умолчанию | Диапазон | Описание |
|----------|--------------|----------|----------|
| `expansion_speed` | 0.1 | 0.01–1.0 | Скорость расширения контейнера |
| `compression_speed` | 0.1 | 0.01–1.0 | Скорость сжатия контейнера |
| `heating_rate` | 0.05 | 0.01–0.5 | Скорость увеличения скоростей частиц |
| `cooling_rate` | 0.05 | 0.01–0.5 | Скорость уменьшения скоростей частиц |

**Влияние параметров**:
- При нагреве: `v_new = v * (1 + heating_rate)`
- При охлаждении: `v_new = v * (1 - cooling_rate)`
- При расширении: `width_new = width + expansion_speed`
- При сжатии: `width_new = width - compression_speed`

## Параметры частиц

Свойства газовых частиц.

| Параметр | По умолчанию | Диапазон | Единицы | Описание |
|----------|--------------|----------|---------|----------|
| `count` | 200 | 1–1000 | шт. | Количество частиц |
| `radius` | 5 | 1–50 | px | Радиус частицы |
| `mass` | 1 | 1–100 | а.е.м. | Масса частицы |
| `initial_velocity` | 5 | 1–50 | px/шаг | Начальная скорость |

**Примечания**:
- Большое количество частиц (`count > 500`) может замедлить симуляцию
- Радиус влияет на частоту столкновений
- Масса влияет на распределение скоростей и передачу импульса

## Физические параметры

### Гравитация

Параметры гравитационного поля для моделирования распределения Больцмана.

| Параметр | По умолчанию | Диапазон | Единицы | Описание |
|----------|--------------|----------|---------|----------|
| `gravity_enabled` | `false` | `true`/`false` | — | Включить гравитацию |
| `gravity_acceleration` | 9.8 | 0.1–100 | px/шаг² | Ускорение свободного падения |

**Примечания**:
- Гравитация направлена вниз (положительное направление оси Y)
- Используется для проверки барометрической формулы

### Броуновское движение

Параметры для моделирования броуновского движения.

| Параметр | По умолчанию | Значения | Описание |
|----------|--------------|----------|----------|
| `brownian_enabled` | `false` | `true`/`false` | Включить броуновский режим |
| `brownian_type` | `"single_large"` | `single_large`, `multi_track` | Тип эксперимента |
| `brownian_mass_ratio` | 10.0 | 1–100 | Отношение массы броуновской частицы к обычной |
| `brownian_radius_ratio` | 3.0 | 1–10 | Отношение радиуса броуновской частицы к обычной |

**Типы эксперимента**:
- `single_large` — одна крупная частица среди мелких
- `multi_track` — отслеживание нескольких частиц

## Потенциалы взаимодействия

### Леннард-Джонс

Классический потенциал для моделирования ван-дер-ваальсовых сил.

```math
U(r) = 4\varepsilon\left[\left(\frac{\sigma}{r}\right)^{12} - \left(\frac{\sigma}{r}\right)^6\right]
```

| Параметр | По умолчанию | Диапазон | Единицы | Описание |
|----------|--------------|----------|---------|----------|
| `enabled` | `false` | `true`/`false` | — | Включить потенциал |
| `epsilon` | 1.0 | 0.01–10 | у.е. | Глубина потенциальной ямы |
| `sigma` | 10.0 | 1–50 | px | Характерный размер (расстояние нулевого потенциала) |
| `cutoff` | 2.5 | 1.5–5.0 | σ | Радиус обрезания в единицах σ |

**Физический смысл параметров**:
- `epsilon` — энергия связи в минимуме потенциала
- `sigma` — эффективный диаметр молекулы
- `cutoff` — на расстояниях > cutoff·σ взаимодействие не учитывается

### Морзе

Потенциал для моделирования химической связи.

```math
U(r) = D_e\left[1 - e^{-a(r-r_e)}\right]^2
```

| Параметр | По умолчанию | Диапазон | Единицы | Описание |
|----------|--------------|----------|---------|----------|
| `enabled` | `false` | `true`/`false` | — | Включить потенциал |
| `D_e` | 1.0 | 0.1–10 | у.е. | Энергия диссоциации |
| `a` | 0.5 | 0.1–5.0 | 1/px | Параметр «жёсткости» |
| `r_e` | 15.0 | 5–50 | px | Равновесное расстояние |

**Физический смысл параметров**:
- `D_e` — энергия, необходимая для разрыва связи
- `a` — определяет ширину потенциальной ямы
- `r_e` — расстояние в минимуме потенциала

### DLVO

Потенциал для коллоидных систем (Дерягина-Ландау-Вервея-Овербека).

| Параметр | По умолчанию | Диапазон | Единицы | Описание |
|----------|--------------|----------|---------|----------|
| `enabled` | `false` | `true`/`false` | — | Включить потенциал |
| `A_hamaker` | 1.0 | 0.01–10 | у.е. | Константа Гамакера (притяжение) |
| `psi` | 0.025 | 0.001–0.1 | В | Поверхностный потенциал |
| `kappa` | 10.0 | 1–100 | 1/px | Обратная длина Дебая |
| `epsilon_r` | 80.0 | 1–100 | — | Относительная диэлектрическая проницаемость |

**Физический смысл параметров**:
- `A_hamaker` — сила ван-дер-ваальсова притяжения
- `psi` — заряд поверхности частицы
- `kappa` — характерная длина экранирования (1/κ)
- `epsilon_r` — свойство среды (80 для воды)

## Режимы симуляции

Доступные режимы изменения состояния системы.

| Режим | Код | Описание | Изменяемые параметры |
|-------|-----|----------|---------------------|
| Выключен | `OFF` | Без изменений | — |
| Нагрев | `heat` | Увеличение температуры | Скорости частиц ↑ |
| Охлаждение | `freeze` | Уменьшение температуры | Скорости частиц ↓ |
| Расширение | `expansion` | Увеличение объёма | Ширина контейнера ↑ |
| Сжатие | `compression` | Уменьшение объёма | Ширина контейнера ↓ |
| Нагрев + расширение | `heat_expansion` | Комбинированный процесс | Скорости ↑, ширина ↑ |
| Нагрев + сжатие | `heat_compression` | Комбинированный процесс | Скорости ↑, ширина ↓ |
| Охлаждение + расширение | `cool_expansion` | Комбинированный процесс | Скорости ↓, ширина ↑ |
| Охлаждение + сжатие | `cool_compression` | Комбинированный процесс | Скорости ↓, ширина ↓ |

**Специальные режимы**:

| Режим | Описание | Эффект |
|-------|----------|--------|
| Изолированная система | Блокировка изменений энергии и объёма | Для проверки законов сохранения |
| Старт из угла | Начальное размещение частиц в углу | Для демонстрации роста энтропии |
| Броуновский | Выделение крупной частицы | Для изучения броуновского движения |
| Гравитация | Включение поля тяжести | Для проверки распределения Больцмана |

## Единицы измерения

В симуляции используются условные единицы (у.е.) для упрощения вычислений.

| Величина | Единица | Примечание |
|----------|---------|------------|
| Длина | px (пиксели) | 1 px ≈ произвольный масштаб |
| Время | шаг симуляции | определяется параметром `dt` |
| Масса | а.е.м. (атомные единицы) | относительная масса |
| Энергия | у.е. | $E = \frac{1}{2}mv^2$ |
| Температура | у.е. | $T = \frac{m\langle v^2 \rangle}{2k_B}$ |
| Давление | у.е. | из переданного импульса стенкам |

**Постоянная Больцмана**: в симуляции принята $k_B = 1$ для упрощения формул.
